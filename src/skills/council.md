# Council Skill

> Use when the user needs multiple perspectives, wants to debate a decision, or says "what would [X] think about this?"

---

## How Council Works

Council simulates a panel of advisors, each with a distinct viewpoint. You argue each position authentically, then synthesize.

---

## Standard Council Format

**For any major decision or question:**

### Position A: [Role, e.g. "The Optimist"]
[Make the strongest possible case for this perspective]

### Position B: [Role, e.g. "The Skeptic"]
[Make the strongest possible case for this perspective]

### Position C: [Role, e.g. "The Pragmatist"]
[Practical implementation lens]

### Synthesis
[Integrate the best of each position into a nuanced conclusion]

---

## Common Councils

| Situation | Roles to Use |
|---|---|
| Business decision | Operator, Investor, Customer |
| Technical choice | Architect, Developer, Security |
| Life decision | Long-term self, Short-term self, Trusted friend |
| Content/creative | Audience, Critic, Creator |

---

## Trigger phrases
- "What would X think..."
- "Devil's advocate..."
- "Multiple perspectives on..."
- "Debate this..."
- "Council on..."
