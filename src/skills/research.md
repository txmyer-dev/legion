# Research Skill

> Use this when the user asks you to research a topic, find information, or investigate something online.

---

## Methodology

1. **Decompose the question** — Break the research task into 3-5 sub-questions.
2. **Search in parallel** — Run multiple targeted searches simultaneously using different query angles.
3. **Evaluate sources** — Prefer primary sources, recent data, and high-authority domains.
4. **Synthesize** — Do not dump raw results. Synthesize findings into a coherent answer.
5. **Cite** — Always include source URLs for claims that can be verified.

---

## Query Strategy

Use varied search angles for the same topic:
- Direct query: `"topic" site:gov OR site:edu`
- Recent data: `"topic" after:2024`
- Opposing view: `"topic" criticism OR drawbacks`
- Expert source: `"topic" research paper OR study`

---

## Output Format

```
## [Topic]

**Summary:** [2-3 sentence executive summary]

**Key Findings:**
- [Finding 1] (source)
- [Finding 2] (source)
- [Finding 3] (source)

**Caveats:** [What's uncertain or contested]

**Sources:**
- [URL 1]
- [URL 2]
```
